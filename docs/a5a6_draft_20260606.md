## A5 — name the three advantage functions in §3, restate unlinkability in advantage form

**(a) Unforgeability** (after 03-preliminaries.tex:421): append
> We write $\mathsf{Adv}^{\mathsf{unf}}_{\AC}(\mathcal A)$ for the probability that $\mathcal A$ wins this experiment; the scheme is unforgeable if it is $\negl(\lambda)$ for every PPT $\mathcal A$.

**(b) Anonymity** (replace the closing sentence at 03:461-468): 
> The adversary outputs $b'$. We write $\mathsf{Adv}^{\mathsf{anon}}_{\AC}(\mathcal A)=\bigl|\Pr[b'=b]-\tfrac12\bigr|$ for the anonymity advantage; the scheme is anonymous if it is $\negl(\lambda)$ for every PPT $\mathcal A$.

**(c) Unlinkability** (replace the ≈_c formulation at 03:487-495 with a b'=b game, mirroring BDEC AdvUNL):
> Consider the experiment in which the challenger samples $b\leftarrow\{0,1\}$ and generates a transcript pair $(T_0,T_1)$ from the \emph{same} honest user if $b=0$, or from two \emph{independent} honest users if $b=1$, in both cases with $\mathsf{Leak}(\mathsf{stmt}_0)=\mathsf{Leak}(\mathsf{stmt}_1)$ and no public linking tag. The adversary outputs $b'$. We write $\mathsf{Adv}^{\mathsf{unlink}}_{\AC}(\mathcal A)=\bigl|\Pr[b'=b]-\tfrac12\bigr|$; the scheme is unlinkable if this is $\negl(\lambda)$ for every PPT $\mathcal A$. When the statements intentionally contain a public linking tag, linkability is allowed only as determined by that tag.

Effect: the symbols $\mathsf{Adv}^{\mathsf{unf/anon/unlink}}$ used in the §055 theorem (eqs. adv-unf/anon)
are now defined; the $2\epsilon_{\mathrm{ZK}}$ unlinkability bound now multiplies a quantity that exists in
advantage form (the b'=b game over the two challenge transcripts, hence the simulator is applied twice).

## A6 — ε_AIR as a probability, not a premise-failure indicator (055-security.tex:137-138)

Current: "$\epsilon_{\mathrm{AIR}}$ is the constraint-level soundness error of the Griffin AIR, carried as
the open premise T2" — summed additively with genuine probabilities ε_KS, ε_EUF, ε_ZK, which is a
type error (a premise is not a probability).

Proposed:
> The term $\epsilon_{\mathrm{AIR}}=\Pr[\text{the Griffin AIR accepts a witnessed transition not equal to the reference permutation}]$ is the AIR's constraint-level soundness error, a genuine probability; premise~(T2) is the open, empirically-supported but unproved claim (Appendix~\ref{app:proofs}) that $\epsilon_{\mathrm{AIR}}$ is negligible.

Optional matching tweak at the theorem premise (055:117-119): "premise (T2), that the
Griffin-$\mathbb{F}_p^{192}$ AIR admits only the reference permutation up to a negligible soundness error
$\epsilon_{\mathrm{AIR}}$" — so T2 and ε_AIR name the same object.
